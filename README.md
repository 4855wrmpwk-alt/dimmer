Управление светом через диммер
Описание программы
Программа для управления яркостью света через диммер с графическим интерфейсом. Поддерживает режим симуляции для тестирования и режим реального устройства для работы с физическим оборудованием.

Основные функции
Управление светом
Включение/выключение света одной кнопкой

Плавная регулировка яркости от 0% до 100%

Предустановки яркости:

Ночник (10%)

Чтение (60%)

Работа (80%)

Максимум (100%)

Авторегулировка - автоматическое изменение яркости

Режимы работы
Режим симуляции - работа без физического устройства

Режим реального устройства - подключение через COM-порт (требует установки pyserial)

Дополнительные возможности
Сохранение настроек - автоматическое сохранение последних параметров

Расписание - установка времени включения/выключения

Визуализация - графическое отображение текущей яркости

Журнал событий - запись всех действий программы

Требования
Python 3.x

Tkinter (обычно входит в стандартную поставку Python)

Для работы с реальным устройством:
Установленный модуль pyserial (pip install pyserial)

Физическое устройство диммера с поддержкой Serial-интерфейса

Установка и запуск
Способ 1: Запуск через Python
Сохраните код в файл dimmer_controller.py

Запустите командой:

text
python dimmer_controller.py
Способ 2: Создание исполняемого файла
Установите PyInstaller:

text
pip install pyinstaller
Создайте EXE-файл:

text
pyinstaller --onefile --windowed dimmer_controller.py
Запустите файл из папки dist

Графическое отображение текущей яркости света

Журнал событий с временными метками

Программа автоматически сохраняет настройки в файл dimmer_settings.json:

Текущая яркость

Состояние (включено/выключено)

Выбранный режим работы

Порт подключения

Расписание

Время автоматического выключения света

Технические детали
Формат команд
В режиме реального устройства программа отправляет команды в формате:

P1B050 - включить свет с яркостью 50%

P0B100 - выключить свет (яркость 100% сохраняется)

Файлы программы
dimmer_controller.py - основной файл программы

dimmer_settings.json - файл сохраненных настроек (создается автоматически)

Устранение неполадок
Проблема: Не открывается окно программы
Решение:

Проверьте установку Python: python --version

Убедитесь, что Tkinter установлен: python -m tkinter

Переустановите Python с включенной опцией "tcl/tk and IDLE"

Проблема: Ошибка "No module named 'serial'"
Решение:

Для режима симуляции - ошибка не критична

Для режима реального устройства установите модуль:

text
pip install pyserial
Проблема: Не сохраняются настройки
Решение:

Проверьте права на запись в папку с программой

Убедитесь, что файл dimmer_settings.json не открыт в другой программе

Безопасность
Программа работает полностью локально

Не требует подключения к интернету

Не собирает пользовательские данные

Все настройки сохраняются только на локальном компьютере

Лицензия
Программа распространяется бесплатно. Может использоваться в личных и образовательных целях.

Поддержка
При возникновении проблем:

Проверьте, что все требования выполнены

Запустите программу из командной строки для просмотра ошибок

Убедитесь, что Python и Tkinter установлены корректно

Советы по использованию
Для тестирования:
Используйте режим симуляции для знакомства с функционалом программы без физического устройства.

Для реального использования:
Подключите диммер к компьютеру через COM-порт

Установите модуль pyserial

Переключитесь в режим реального устройства

Выберите правильный COM-порт и подключитесь
