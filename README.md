📋 Требования

bash
✅ Python 3.6 или выше
✅ Tkinter (входит в стандартную поставку)
✅ Для реального устройства: pyserial
🔧 Установка

Клонируйте репозиторий
bash
git clone https://github.com/yourusername/light-dimmer-controller.git
cd light-dimmer-controller
Установите зависимости
bash
pip install pyserial  # Только для работы с реальным устройством
Запустите приложение
bash
python light_dimmer.py
🎯 Альтернативный способ

Скачайте light_dimmer.py
Дважды кликните файл (если Python ассоциирован)
Или запустите через командную строку

📍 Подготовка оборудования

text
Физическое устройство → COM-порт → Приложение
       ↓                    ↓           ↓
   Световой диммер     USB-кабель   Контроллер
🔄 Шаги подключения

Выберите режим → "Реальное устройство"
Подключите оборудование через COM-порт
Выберите порт из выпадающего списка
Нажмите "Подключить"
Начните управление
⚠️ Если порты не отображаются

bash
# Windows: проверьте диспетчер устройств
# Linux: проверьте права доступа
ls -l /dev/tty*
# macOS: проверьте подключение

📡 Протокол связи

python
# Формат команды для реального устройства
"P{state}B{brightness:03d}"
# Пример: P1B075 = Включить с яркостью 75%
💾 Формат файла настроек

json
{
  "brightness": 75,
  "is_on": true,
  "simulation_mode": false,
  "port": "COM3",
  "schedule": {
    "on_time": "18:00",
    "off_time": "23:00"
  }
}
